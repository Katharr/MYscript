# 梦幻西游 · 时空　辅助助手

截屏 + 图像识别 + 拟人化模拟点击的辅助工具，带图形界面。当前已实现 **秒装备（市场捡漏）**，
架构做了模块化，方便以后加「自动刷副本 / 师门 / 抓鬼」等功能。

> ⚠️ **封号风险**：使用任何第三方脚本都违反《梦幻西游》用户协议，可能被封号（含永封）。
> 请务必用**小号**测试，自负风险。本工具仅供学习交流。

---

## 一、怎么用（双击即可）

1. 装 [Python 3.10+](https://www.python.org/downloads/)（安装时勾选 “Add Python to PATH”）。
2. **双击 `启动.bat`**。首次会自动安装依赖，然后弹出图形界面。
3. 界面里：
   - 先点 **「标定 / 加装备」**：打开游戏到市场界面，按弹出的小窗口提示，用鼠标框出
     「列表 / 刷新 / 购买 / 确认」按钮位置，再框出你想抢的装备。完成后回到界面点 **「刷新配置」**。
   - 默认是 **演练模式**（只识别、不下单，安全）。点 **「开始秒装备」** 跑一下，看右侧日志和
     `captures/` 里的命中截图，确认识别准不准。
   - 确认没问题后，把 **「实战模式」** 开关打开，再「开始秒装备」，命中才会真正下单。
4. 想停就点 **「停止」**；紧急情况把鼠标猛甩到屏幕左上角也能刹停。

## 二、底层 + 拟人化设计（针对“别被封号”）
- 鼠标输入默认走 Windows `SendInput`（user32）——用户态最底层的标准输入接口，比高层封装更难被规则识别。
- 鼠标走贝塞尔曲线、有加减速、不瞬移；落点随机偏移；按下/抬起与各种间隔都加随机抖动；偶尔“走神”停顿。
- 诚实说：真正“完全测不到”要硬件级(KMBox/Arduino)或驱动级，是另一量级工程且自身也可能被风控盯。
  本方案是性价比折中，**不保证 100% 安全**。

关于价格：现为 **命中即抢**，不做 OCR 比价（模板匹配认不了“任意低于 X 价”）。
建议模板**只框装备本身**，别框价格。

## 三、项目结构（模块化，便于扩展）
```
myScript/
├─ 启动.bat              # 一键入口（双击）
├─ start.py             # 装依赖 + 启动 GUI
├─ requirements.txt
├─ config.json          # 配置（标定后自动生成；tasks.<任务名> 命名空间）
├─ config.example.json  # 配置示例
├─ templates/           # 标定时保存的装备模板图
├─ captures/            # 命中存证截图
└─ mhxy/                # 主包
   ├─ core/             # 通用基础设施（与玩法无关）
   │  ├─ config.py      #   配置读写
   │  ├─ window.py      #   找窗口 / 截图 / 坐标换算
   │  ├─ vision.py      #   模板匹配 / 图片读写
   │  ├─ input.py       #   SendInput 底层 + 拟人化鼠标
   │  ├─ context.py     #   TaskContext（打包窗口/鼠标/日志/停止信号给任务）
   │  └─ runner.py      #   后台线程跑任务 + 日志队列
   ├─ tasks/            # 可插拔任务
   │  ├─ base.py        #   Task 基类 + 注册表
   │  └─ sniper.py      #   秒装备任务
   ├─ tools/
   │  └─ calibrate.py   #   标定向导（GUI 以子进程调用）
   └─ gui/
      ├─ theme.py       #   配色/字体/圆角（改这里整体换肤）
      └─ app.py         #   主界面（侧边导航 + 分页）
```

### 想加一个新功能（比如“自动师门”）
1. 在 `mhxy/tasks/` 新建 `shimen.py`，写个 `class ShimenTask(Task)`，实现 `run(ctx)`（循环里勤查 `ctx.should_stop()`），用 `@register` 注册。
2. 在 `mhxy/tasks/__init__.py` 里 `from . import shimen`。
3. 在 `mhxy/gui/app.py` 加一个对应页面（可仿照 `SniperPage`）和导航项。
基础设施（截图/识别/拟人化点击）直接复用 `ctx`，不用重写。

## 四、调参（设置页 + `config.json`）
- 设置页可改：游戏窗口标题关键字、鼠标输入后端。
- `config.json` 里：
  - `tasks.sniper.loop.match_threshold`：识别相似度。漏抢调低(0.80)，误抢调高(0.90)。
  - `tasks.sniper.loop.refresh_interval_sec`：刷新间隔（会自动±抖动）。
  - `humanize.*`：拟人化参数（落点随机、移动平滑度、抖动比例、走神概率），已给稳妥默认值。
  - `window_offset`：点击整体偏移时补 `[dx, dy]` 修正。
